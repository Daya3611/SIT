# AgriPayChain - Page Structure

This document outlines the routing and page structure of the AgriPayChain platform.

## Public Web Experience (Multi-Page)
The public site is now split into dedicated SEO-optimized pages for better performance and clarity.

- `/` - **Home Page**
  - **Cinematic Journey**: Leading with the high-performance video scroll sequence.
  - **Welcome Hero**: Simple introduction to the platform's core premise.
  - **Global Footer**: Quick links to the entire ecosystem.

- `/about` - **About Us**
  - Mission Statement & Platform Philosophy.
  - **Process**: How it works (4-step visual).
  - **Capabilities**: Feature highlights.
  - **Trust**: Technology deep-dive (Blockchain/AI/IoT).

- `/contact` - **Contact & Connect**
  - Interactive CTA portal.
  - Multi-channel support information (Phone/Email/Office).

- `/marketplace` - **Marketplace (Coming Soon)**
  - Placeholder for the direct-to-buyer produce exchange.

## Farmer Dashboard Portfolio
Accessible via the **"Launch App"** button.

- `/dashboard` - **Overview**
- `/dashboard/land` - **Land Records**
- `/dashboard/crop` - **Crop Management**
- `/dashboard/shipment` - **Shipment Tracking**
- `/dashboard/payment` - **Payments & Finance**
- `/dashboard/scheme` - **Scheme Scanner**
- `/dashboard/insights` - **AI Insights & Risk**
